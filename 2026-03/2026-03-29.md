# Tech Log

## Stimulusマニフェストで沼った
【事象】
- JSファイル（liff_controller.js）は存在するし、コードも正しい。
- HTML側で data-controller="liff" も指定している。

**それなのに`connect()`内のログすら出ず、JSが1行も実行されない。**

【原因】
- app/javascript/controllers/index.js（マニフェストファイル）に、新しく作成した liff_controller が登録されていなかった。

**※Stimulusは、この index.js にインポート・登録されていないコントローラーを認識しない。**

【対処】
下記コマンドをターミナルで実行し、マニフェストを更新
```
bin/rails stimulus:manifest:update
```
