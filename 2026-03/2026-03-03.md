## Tech Log

### Rubyのベースイメージとwget
- Ruby:3.3.6の場合：
```
docker run --rm ruby:3.3.6 bash -c "apt-get update > /dev/null 2>&1 && which wget"
# /usr/bin/wget と表示される → wgetがインストールされている
```
- Ruby:4.0.1の場合：
```
docker run --rm ruby:4.0.1 bash -c "which wget"
# /usr/bin/wget  と表示される → wgetがインストールされている
```
