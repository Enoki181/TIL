### Railsアプリ作成コマンド

- RUNTEQのヘルプページにおける `rails new`
  ```
  docker compose run --rm web rails new . -d postgresql -j esbuild --css=tailwind
  ```
  - Dockerで一時的に `web` コンテナを起動して `rails new` を実行
  - `--rm` により実行後コンテナは削除される
  - `-d postgresql`
    - DBをPostgreSQLに指定
  - `-j esbuild`
    - JavaScriptのビルドツールとしてesbuildを使用
  - `--css=tailwind`
    - CSSフレームワークとしてTailwind CSSを使用

### Controller生成コマンド
```
rails g controller hello index
```
実行すると以下が生成(追加)される
- `HelloController` 
- `index` アクション
- `routes.rb` にルーティングが追加

### ERBテンプレート

`html.erb` ではRubyコードを埋め込める。

- `<% ... %>`
  - Rubyコードを実行する（出力はされない）

- `<%= ... %>`
  - Ruby式の結果をテンプレートに出力（Viewに表示）
